Backend Server

Although django offers a wide range of tools to build a complete application, you can use also use django restframework solely as a backend server. By using django as your framework, you can use a more powerful frontend framework such as React or Angular. In order to do this, all you need to do is serialize the information in your django models. Here is a list of the models used in this project and their serialized forms.

Post Model

The serialized data for the Post Model in the Blog App.

Other requests: GET, PUT, & DELETE.

Fields: pk (primary key), post title, post preview, post content, date posted and author
